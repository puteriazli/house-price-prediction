Cara menjalankan project:
1. Buka CMD (Command Prompt)
2. Masuk ke folder backend
3. Ketik Python app.py
4. Buka CMD dijendela baru
5. Masuk ke folder frontend
6. Ketik npm run dev
7. Salin alamat URL yang muncul di CMD
4. Buka browser di computer
5. Jalankan URL/alamat host yang didapat dari CMD
6. Input data sesuai kebutuhan prediksi
7. Klik tombol prediksi
